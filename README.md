# sports-mate

## サイト概要
sports-mateは、様々なスポーツを行うクラブがメンバーを募集することができるサイトです。

### サイトテーマ
スポーツで出会い健康に

### テーマを選んだ理由
自分がスポーツ好きで引っ越した時にスポーツをしたかったが探せなかったため。

### ターゲットユーザ
スポーツが好きで主催しメンバーを募集したい方。活動に参加したい方。

### 主な利用シーン
スポーツ活動をする際、スポーツしたいと思った時、メンバーを集めたい時。
